Font Information URL
Merriweather Bold Font is declared within base.css. The font is used for headings https://fonts.google.com/specimen/Merriweather?selection.family=Merriweather
Merriweather Light Font is declared within base.css. The font is used for sub-headings https://fonts.google.com/specimen/Merriweather?selection.family=Merriweather
Muli Light / Muli Bold Font is declared within base.css. The font is used for all text except headings. https://fonts.google.com/specimen/Muli
